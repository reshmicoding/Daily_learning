Exercise 5 — risk transition (changing of the guard)
High-growth EV startup → mature automotive peer. Rf = 4%, ERP = 5%, Tax = 25%
High-growth WACC
Ke = 4% + 1.8 × 5% = 13.0% | Kd(AT) = 8% × (1−0.25) = 6.0% WACC = 13.0% × 0.85 + 6.0% × 0.15 = 11.95% ≈ 12.0%
Mature WACC (used to discount TV)
Ke = 4% + 1.05 × 5% = 9.25% | Kd(AT) = 5.5% × 0.75 = 4.125% WACC = 9.25% × 0.60 + 4.125% × 0.40 = 7.20%
High-growth WACC
11.95%
Mature WACC
7.20%
You MUST use the mature WACC (7.20%) in the terminal value denominator — not the high-growth WACC. The TV represents the firm in stable state at Year 10+, so its risk profile should reflect the mature firm, not the startup.
Python
rf = 0.04
erp = 0.05

RR = 3%/12% = 25% | FCFF = 112.5M

The moat scenario produces 25% higher terminal value ($2,500M vs $2,000M). This is purely because a higher ROIC requires less reinvestment to achieve the same 3% growth — freeing up more cash flow. When ROIC = WACC, growth creates zero incremental value.
Python
ebit_at = 150
gn = 0.03
wacc = 0.075

scenarios = {"Moat fades (ROIC=WACC)": 0.075, "Moat persists (ROIC=12%)": 0.12}

for name, roic in scenarios.items():
    rr = gn / roic
    fcff = ebit_at * (1 - rr)
    tv = fcff / (wacc - gn)
    print(f"{name}: RR={rr:.1%}, FCFF=${fcff:.1f}M, TV=${tv:,.0f}M")

# percentage diff
tv1, tv2 = [ebit_at*(1-gn/r)/(wacc-gn) for r in scenarios.values()]
print(f"\nPremium from moat: {(tv2/tv1 - 1):.1%}")
Exercise 7 — reinvestment mismatch paradox
Year 5: Net CapEx = 0 but gₙ = 3.5% projected. EBIT(1-t)₅ = $400M · ROIC = 10% · WACC = 8%
Structural flaw: Net CapEx = 0 (CapEx = Depreciation) means no reinvestment. But 3.5% growth requires reinvestment. These two assumptions are internally inconsistent — zero investment cannot generate positive growth.
Corrected Year 6 using Damodaran linkage
EBIT(1-t)₆ = EBIT(1-t)₅ × (1 + gₙ) = 400 × 1.035 = $414M RR = gₙ / ROIC = 3.5% / 10% = 35% FCFF₆ = 414 × (1 − 0.35) = $269.1M TV₅ = 269.1 / (0.08 − 0.035) = $5,980M
❌ Flawed model (Net CapEx=0)
$9,100M
FCFF₅×1.035/spread = inflated
✓ Corrected model
$5,980M
internally consistent
Python

gn = 0.035
roic = 0.10
wacc = 0.08

ebit6 = ebit5 * (1 + gn)

The finite life haircut is ~32.3% — nearly a third of value is lost vs a perpetuity assumption. The annuity factor (0.677) captures the present value of 15 years of growing cash flows at WACC.
Python

gn = 0.02
t = 15

Exercise 9 — currency inflation trap
Argentine subsidiary valued in ARS. Local inflation = 20%. ARS risk-free = 22%. Colleague wants gₙ = 4% (USD cap). Correct?
Peer is wrong. The growth cap is currency-specific. A 4% USD cap applies only to USD-denominated cash flows. ARS cash flows must use the ARS risk-free rate as the growth ceiling.
Correct growth cap for ARS valuation
gₙ(ARS) ≤ Risk-free rate (ARS) = 22% Using 4% would massively understate the terminal value because ARS cash flows reflect 20% inflation — they must grow at ~20% just to stay flat in real terms.
❌ Peer's approach: USD cap 4% in ARS
Wrong currency ceiling applied — deflates TV by ~4x
✓ Correct: ARS ceiling = 22%
Nominal ARS risk-free = inflation proxy + real rate
The iron law: gₙ ≤ risk-free rate of the SAME currency. ARS has 22% RF because it embeds 20% expected inflation. A 22% growth cap in ARS ≈ 2% real, which is consistent with the USD cap of ~4%.

Neo-Crafts Ltd — mini valuation project
Year 5 EBIT = $20M · Tax = 30% · Rf = 4% · ERP = 5.5% · Beta_stable = 1.0 · Kd = 6% · D/C = 30% · gₙ = max allowed · ROIC = WACC (competitive equilibrium)
1 — stable WACC
Ke = 4% + 1.0 × 5.5% = 9.5% Kd(AT) = 6% × (1 − 0.30) = 4.2% WACC = 9.5% × 0.70 + 4.2% × 0.30 = 7.91%
2 — stable gₙ and RR
gₙ = Rf = 4% (Damodaran maximum) ROIC = WACC = 7.91% (competitive equilibrium assumed) RR = gₙ / ROIC = 4% / 7.91% = 50.6%
3 — EBIT(1-t) and FCFF₆
EBIT(1-t) = 20 × (1 − 0.30) = $14M FCFF₆ = 14 × (1 + 0.04) × (1 − 0.506) = $7.19M
4 — TV₅
TV₅ = FCFF₆ / (WACC − gₙ) = 7.19 / (0.0791 − 0.04) = $183.8M
5 — % of enterprise value from TV
PV(FCFF₅) = 42M (given) PV(TV) = TV₅ / (1 + WACC_hg)^5 = 183.8 / (1.12)^5 = $104.3M Total EV = 42 + 104.3 = $146.3M TV % of EV = 104.3 / 146.3 = 71.3%

TV contributes 71.3% of enterprise value — typical for a stable-growth firm. Even though gₙ = WACC (ROIC = WACC), growth still creates value through positive FCFF — the zero-value insight applies to incremental reinvested capital, not the base earnings.
Python — complete Neo-Crafts solution
ebit_y5 = 20
tax = 0.30
rf = 0.04
erp = 0.055
beta_s = 1.0
kd = 0.06
dc = 0.30
ec = 1 - dc
wacc_hg = 0.12      # given high-growth WACC
pv_explicit = 42    # given

ke = rf + beta_s * erp
kd_at = kd * (1 - tax)
wacc_s = ke * ec + kd_at * dc

gn = rf             # max allowed (Damodaran cap)
roic = wacc_s       # ROIC = WACC (competitive equilibrium)
rr = gn / roic

ebit_at = ebit_y5 * (1 - tax)
fcff6 = ebit_at * (1 + gn) * (1 - rr)
tv5 = fcff6 / (wacc_s - gn)

pv_tv = tv5 / (1 + wacc_hg)**5
total_ev = pv_explicit + pv_tv
tv_pct = pv_tv / total_ev

print(f"Stable WACC:  {wacc_s:.2%}")
print(f"gn (capped):  {gn:.2%}")
print(f"RR:           {rr:.1%}")
print(f"FCFF6:        ${fcff6:.2f}M")
print(f"TV5:          ${tv5:.1f}M")
print(f"PV(TV5):      ${pv_tv:.1f}M")
print(f"Total EV:     ${total_ev:.1f}M")
print(f"TV % of EV:   {tv_pct:.1%}")
